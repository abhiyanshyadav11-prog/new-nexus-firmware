# Nexus Firmware

AI wearable pendant firmware for XIAO ESP32-S3.

## Hardware

* XIAO ESP32-S3
* INMP441 Microphone
* WS2812 RGB LED
* Vibration Motor
* Battery + Charging Circuit

## Features

* BLE Framework
* Audio Capture (I2S)
* Battery Monitoring
* RGB LED Control
* Haptic Feedback
* Power Management

## Build Status

✅ Compiles successfully on ESP-IDF v5.2.7
